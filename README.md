# Harsh M. Parikh - Terminal Portfolio

A highly interactive, 3D-accelerated portfolio designed to showcase engineering projects, technical architecture, and cybersecurity mitigation strategies. 

## ⚡ Architecture & Tech Stack

This environment is built for maximum performance and tactile user interaction, utilizing modern web standards:

* **Core:** Next.js (App Router), React, TypeScript
* **Styling:** Tailwind CSS, Glassmorphism UI
* **3D Rendering:** Three.js, React Three Fiber, Drei
* **Physics & Motion:** Framer Motion (Spring physics, scroll-linked parallax)
* **Icons:** React Icons (FontAwesome, Simple Icons)

## 🚀 Key Features

* **Zero-Trust Boot Sequence:** Custom cinematic decryption pre-loader.
* **Root Terminal Access:** Hidden interactive command-line interface triggered by `Ctrl + K`.
* **Magnetic UI:** Hardware-accelerated cursor with physical "pull" and spring-release mechanics on interactive elements.
* **Cryptographic Text Reveal:** Scroll-triggered text decryption effects.
* **Deep Parallax Camera:** Scroll-linked 3D camera that flies through the WebGL background environment.
* **Vault Lock Modals:** Secured, full-screen case study expansions for technical projects.

## ⚙️ Local Deployment

To boot the system locally, clone the repository and run the following terminal commands:

\`\`\`bash
# 1. Install dependencies
npm install

# 2. Boot the development server
npm run dev
\`\`\`
The environment will initialize at `http://localhost:3000`.
